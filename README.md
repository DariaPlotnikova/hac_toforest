## Инструкция по установке

* Установите virtualenv
* Выполните ``virtualenv venv`` или на винде - ``workon venv``
* Активируйте виртуал окружение
```
source venv/bin/activate

# windows
cd venv/Scripts
activate.bat
```

* Установи зависимости
```
pip install -r requirements.txt
```

## Выйти из venv
```
deactivate
```
